Лабораторная работа №2
Terraform как инструмент для декларативного описания инфраструктуры

В данной лабораторной работе создал 4 ВМ в двух подсетях , установил на них nginx, создал балансировщик, развернул кластер mySQL

1. Cоздал файл variables.tf в которых описал переменные, в том числе переменную host_count, через которую можно передать количество хостов.
2. В файле network.tf создал сеть и три подсети.
3. В файле wp-app.tf создаю n машин и размещаю их в двух подсетях (wp-subnet-a и wp-subnet-b)
4. В файле lb.tf создаю балансировщик
5. В файле db.tf создаю кластер mySQL 
6. На git не загрузил файл с переменными wp.auto.tfvars, его содержимое следующее
    yc_cloud  = "id_cloud"
    yc_folder = "id_folder"
    yc_token = "id IAM токена, созданного для сервисной учетной записи"
    db_password = "Пароль админа БД mySQL"

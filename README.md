# IaC_public
В папке terraform манифест, который разворачивает две виртуальные машины, балансировщик и кластер БД.
В папке ansible playbook, который донастраивает эти две машины.
На них устанавливается apache, wordpress, заливается настройка apache.
Перезапускается apache (поскольку на прошлом занятии, я на данные машины устанавливал nginx, дополнительно останавливаю nginx).

# IaC_public
Лабораторная работа №4
Переиспользование кода Ansible и работа с ролями

В папке terraform манифест, который разворачивает две виртуальные машины, балансировщик и кластер БД (подробное описание работы terraform в файле README.md в папке terraform).
В папке ansible playbook, который донастраивает эти две машины (в папке role содержится роль, которая делает то же самое).
На обе виртуальные машины устанавливается apache (и ряд других пакетов), wordpress, заливается конфигурация apache и wordpress.
Перезапускаю apache (поскольку на прошлом занятии, я на данные машины устанавливал nginx, дополнительно останавливаю nginx).
